# Desafio_03_JS

## 🔗 Live Demo
(https://rodgarhel.github.io/Desafio_03_JS/)

## Description
A responsive web design project focused testing different functions with javaScript. 
### 4_colores:
  Testing events  that change an attribute using onClick and keydown events.
### pintar:
  testing how to paint a DOM element using onClik event with the restriction of calling only once the DOM element and using addEventListener. 
### 1_funcion  and 2_arrow javascript files:
  Here the idea was to change the type of declaration of  a function, maintaining  it's functionality. 


## 🛠 Technologies Used

- HTML5
- CSS3
- Media Queries
- JavaScript
